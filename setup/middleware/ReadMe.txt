1. To run this app, open command prompt at bin directory and run below command

> start.bat then press enter

2. To see middleware logs, go below path

>C:\middleware\logs\logFile.{date}.log

3. For any configuration changes, goto C:\middleware\conf\application.properties file, if you want to change any configuration stop the server then edit the file and then start the server.

4. To stop this app, open command prompt at bin directory and run below command

> stop.bat then press enter
